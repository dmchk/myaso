# Myaso

Myaso [ˈmʲæ.sə] is a morphological analysis and synthesis library,
written in Ruby.

![Myaso](/eveel/myaso/raw/develop/myaso.jpg)

## Installation

Add this line to your application's Gemfile:

    gem 'myaso'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install myaso
